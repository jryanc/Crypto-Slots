Crypto Slots Launcher

Open Crypto Slots App

How to Run the Application:

  1. Open Command Prompt (cmd)
  2. Navigate to the server directory:
    cd C:\Users\jryan\Desktop\crypto-slots\server
  3. Install server dependencies (if not already installed):
    npm install
  4. Start the server:
    npm run dev
  5. Open another Command Prompt window
  6. Navigate to the client directory:
    cd C:\Users\jryan\Desktop\crypto-slots\client
  7. Install client dependencies (if not already installed):
    npm install
  8. Start the client:
    npm start
  9. Once both server and client are running, click the button above to open the app

Troubleshooting:

  • Make sure both server and client are running before clicking the button
  • If you see "Cannot GET /", the server might not be running correctly
  • Check the command prompts for any error messages
  • The client might open on a different port. Check the command prompt output for the correct URL
